# 6502e

A 6502 emulator written purely in C99.

## Building

You can build 6502e just like any other package: 
```bash
mkdir build
cd build
../configure
make
```

The resulting binary should be located at `build/src/6502e`.

To install 6502e, run `sudo make install`.

## Contributing
